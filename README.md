# TrackAnalyzer

**TrackAnalyzer** is a native macOS application for hikers, backpackers, climbers, and outdoor enthusiasts who want to analyze GPS tracks in greater detail than is available in many mapping applications.

TrackAnalyzer imports GPX track files and calculates detailed statistics.  Map points can be edited on the main map and elevation can be edited directly from the Elevation Plot.

---

## Features

### 📍 GPX Track Analysis
- Import GPX track files from GPS devices and mapping applications.
- View and edit your track on an interactive map.
- Display and edit an elevation profile.
- Change how elevation data is smoothed.

### 📈 Detailed Statistics
TrackAnalyzer calculates a variety of statistics, including:

- Total distance
- Total ascent
- Total descent
- Maximum and minimum elevation
- Moving time
- Stopped time
- Average moving speed
- Average overall speed
- Ascent distance
- Descent distance
- Ascent time
- Descent time
- Summit time
- Ascent and descent pace

### 🖥 Native macOS Application

TrackAnalyzer is designed specifically for macOS using Apple's native technologies to provide a clean, responsive interface.

## Installation

1. Download the latest **TrackAnalyzer.dmg** from the **Releases** page.
2. Open the downloaded DMG file.
3. Drag **TrackAnalyzer** into the **Applications** folder.
4. Launch TrackAnalyzer from your Applications folder.

The application is digitally signed with an Apple Developer ID and notarized by Apple.

## Requirements

- macOS 15 Sequoia or later
